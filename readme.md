## FinanceTrack

FinanceTrack is a full-stack personal finance management application that allows users to track income and expenses, manage budgets, and visualize spending patterns.

## Features
- User Registration & Login
- Password Hashing with bcrypt
- Add Transactions
- Edit Transactions
- Delete Transactions
- Income, Expense, and Balance Summary
- Budget Goal Tracking
- Interactive Charts
- Dark Theme 

## Technologies Used
# Frontend 
- React
- React Router
- Recharts
- CSS
# Backend
- Node.js
- Express.js
- MySQL
- bcrypt

## Installation
# Frontend
cd frontend
npm install
npm start
# Backend
cd backend
npm install
node server.js
